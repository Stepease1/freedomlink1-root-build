# Intellectual Property and Licensing Framework

**Established**: 2026-07-02  
**Governing License**: MIT + Governance Terms

## I. Core Licensing

### 1.1 Primary License
All code and documentation are released under the MIT License:
- Free for any use (commercial or personal)
- Attribution required
- No warranty provided
- Full text in repository LICENSE file

### 1.2 Governance Terms
In addition to MIT terms, all contributions are subject to:
- Governance charter compliance
- Contributor attribution requirements
- Permanent record-keeping
- Cryptographic verification standards

## II. Contributor IP Rights

### 2.1 Ownership
- Contributors retain IP ownership of their work
- License to FreedomLink1 is perpetual and royalty-free
- Contributors grant non-exclusive rights
- FreedomLink1 cannot revoke license

### 2.2 Attribution
- All contributions credited in registries
- Creator name preserved permanently
- Contribution type documented
- Date and method recorded

### 2.3 Derivative Works
- Permitted under MIT license
- Must include original license
- Attribution to original creator required
- Modifications must be indicated

## III. Third-Party IP

### 3.1 External Dependencies
When incorporating external work:
1. License compatibility verified
2. Original license preserved
3. Attribution included
4. Modifications documented

### 3.2 Acceptable Licenses
- MIT
- Apache 2.0
- GPL v3
- BSD (2-Clause and 3-Clause)
- CC0 (Public Domain)
- Other permissive licenses (case-by-case)

### 3.3 License Review
- All new dependencies reviewed
- Council must approve licenses
- Incompatible licenses rejected
- Alternative solutions sought

## IV. Patent Policy

### 4.1 Patent Grant
Contributors grant:
- Perpetual patent license
- Worldwide scope
- Royalty-free
- Non-exclusive
- Cannot be revoked

### 4.2 Patent Defense
- If anyone sues over patents, their license terminates
- Applies only to defensive actions
- Encourages good-faith collaboration

### 4.3 Patent Contributions
- Contributors must disclose known patents
- Patents cannot block use of contributions
- Patent holders must grant free license

## V. Trademark Policy

### 5.1 FreedomLink1 Marks
- "FreedomLink1" is governed collectively
- Used only for official project
- Community should not use in own projects
- Exceptions granted by Council

### 5.2 Individual Attribution
- Contributor names are personal marks
- Individuals control their own name usage
- Cannot be restricted by project
- Should be used with integrity

## VI. Confidentiality

### 6.1 Public Nature
All FreedomLink1 work is public:
- No confidential information accepted
- All records permanent and verifiable
- Cannot be retracted
- Community has access

### 6.2 Contributor Secrets
- Contributors may keep private keys secret
- Private keys never committed
- Public keys published for verification
- Confidentiality of personal data respected

## VII. Dispute Resolution

### 7.1 IP Disputes
Issues resolved through:
1. Informal discussion
2. Mediation by Council
3. Formal arbitration
4. Legal proceedings (last resort)

### 7.2 Remedies
- Removal of infringing material
- Corrected attribution
- Compensation (if warranted)
- Injunctive relief (if necessary)

## VIII. Copyright Notice

**Copyright © 2026 FreedomLink1 Contributors**

All contributors retain copyright to their work while licensing to project.

## IX. Future Licensing Changes

License changes require:
- 80% Council supermajority
- 30-day notice to community
- Opportunity for contributors to opt-out
- New license compatible with existing works
- Retroactive application to prior contributions only with consent

## X. Compliance

All contributors confirm:
- They own or control the IP they contribute
- No third-party IP is infringed
- They have authority to license the work
- They will comply with this framework
